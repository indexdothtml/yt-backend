# HTTP Status Code

## Overview

- 1xx - Informational
- 2xx - Success
- 3xx - Redirection
- 4xx - Client Error
- 5xx - Server Error

## Detailed

The status code is not standarized, some of them are very common but no official standarization of using status code product can use any status code by there choice.

But following are some common used status codes.

- 100 - Continue
- 102 - Processing (If data sent by client is big and if takes more time to process then we can inform client that data is in process.)
- 200 - Ok
- 201 - Created
- 202 - Accepted
- 307 - Temporary Redirected
- 308 - Permanent Redirected
- 400 - Bad Request
- 401 - Unauthorized
- 402 - Payment Required
- 404 - Resource Not Found
- 500 - Internal Server Error
- 504 - Gateway Time Out
